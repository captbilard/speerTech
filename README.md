# API Documentation

### Step 1

- clone this repo
- run `npm install`
- copy the `.env.example` to a `.env` file in the root directory and fill in the necessary variable.
- Run the command below to run migrations

```
    $ npm run migrate
```

- Run the command below to start the app

```
   $ npm run start
```

- Run the command to run all test

```
    $ npm test
```
